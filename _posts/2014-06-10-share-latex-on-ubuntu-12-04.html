Node.js

[code lang=bash]
sudo apt-get install python-software-properties
sudo add-apt-repository ppa:chris-lea/node.js
sudo apt-get update
sudo apt-get install nodejs
node -v
npm -v
[/code]

grunt-cli (grunt running tools, npm install project dependency)

[code lang=bash]
npm install -g grunt-cli
[/code]

Mongdb

[code lang=bash]
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 7F0CEB10
echo 'deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen' | sudo tee /etc/apt/sources.list.d/mongodb.list
apt-get update
apt-get install mongodb-org
update-rc.d mongdb default
/etc/init.d/mongod start
[/code]

texlive

[code lang=bash]
sudo add-apt-repository ppa:texlive-backports/ppa
sudo apt-get update
sudo apt-get install texlive
[/code]

but texlive is 2012, you can install 2013 by install-tlperl script

[code lang=bash]
rm -rf /usr/local/texlive/2013
rm -rf ~/.texlive2013
./install-tl -gui text
[/code]

vim /etc/environments

[code lang=text]
LC_ALL="en.US.UTF-8"
PATH=/usr/local/texlive/2013/bin/i386-linux:$PATH (maybe 64)
[/code]

redis

[code lang=bash]
sudo add-apt-repository -y ppa:rwky/redis
sudo apt-get update
sudo apt-get install -y redis-server
[/code]

sharelatex

[code lang=bash]
git clone https://github.com/sharelatex/sharelatex.git
cd sharelatex
npm install
grunt install
grunt check --force
grunt run
[/code]

if error, rm -rf web/node_modules/bcrypt, then grunt install again.
