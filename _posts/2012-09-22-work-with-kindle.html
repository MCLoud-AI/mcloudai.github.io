1. first install tools:

File to kindle (PC upload file to kindle)

push to kindle (chrome) butter than the office tool

pdfactory (4.65 support win8 , pdf generating tools)

calibre (epub, pdf to mobi / azw3)

2. upload

txt/word files directly use "File to kindle" to upload to kindle

ppt: use pdfactory to generate 6inch pdf, then uploading.

pdf: using calibre generate mobi file, then uploading.

wiki / web : using push to kindle extension in chrome.
