2.2 activation

Now, the windows must be activation. there are two ways: KMS(180 days) slmgr.exe and MAK (permanent). but The KMS server is block. so it can only use MAK key to activate system.  using slui.exe 3 to open the activation windows. if you can not activate it online. you must cut off your network. then the option of activation by phone could appear. choose the location and skype to the 800 number to activate it.
