x is the protocal of image transition, it can transmit the view of program (x client) to the x server(the program for display and operation)

x client(program) <-> x server(display)

x system : xorg xfree86

If you have the x system, it is just display, can not manage the windows.

XDM is the window manager.

gnome kde xfce have more manager program with windows manager.

xfce is good one for this....

If you want install the x server on windows, your server is in unix/linux

1. using the sshd  X11 forward.... (then, all the program  to sshd client)

2. install the x server (xming) on windows

3. runing the graphic program on unix, it will display on your windows system

xfce and gnome are also graphic program ... so it can do this way...
