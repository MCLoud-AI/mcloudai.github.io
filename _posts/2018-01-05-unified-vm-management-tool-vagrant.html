Vagrant is implemented by Ruby for support all kinds of VM: vitural box, VMware, Hyper-v, Xen and KVM

Configuration file in vagrantFile (Ruby File)

vagrant up

this command will run the script in the VagrantFile

Vagrant status / global-status

This command will show the running VMs

  1. vagrant ssh

This command can attach to VM

vagrant port

This command shows the port mapping from VM to Host
