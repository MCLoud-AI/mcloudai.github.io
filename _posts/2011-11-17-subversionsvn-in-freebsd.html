Our experiment lab need the svn server, so i install the svn in lab server ...

  1. install pkg_add / apt-get install subversion

  2. svnadmin create /repo/project

  3. edit conf in project folder

cancel # (make sure the word in the first position of line)

anon-aceess = read / none

anon-access =write

password-db =passwd

  1. add user passwd in passwd

[user]

xxx = xxx

============= FreeBSD=================

  1. before start

svnserve_enable="YES"
svnserve_flags="-d --listen-port=3690 --listen-host 192.168.1.1"   (if not listen-port 3690  the svn server only listen to ipv6)
svnserve_data="/usr/local/subversion"

  1. then  start

/usr/local/etc/rc.d/svnserve start   -------- this method could not start the server up .... you may use the flowing method ....

svnserve -d -r /usr/local/subversion/ --listen-port=3690 --listen-host 192.168.1.1

============= Ubuntu =======================

svnserve -d --foreground -r /repo --listen-port=3690 --listen-host 192.168.1.1

add script to /etc/init.d/
